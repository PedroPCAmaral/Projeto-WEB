Sobre o Projeto

Este projeto foi desenvolvido como parte da avaliação prática de HTML e CSS, com foco na criação de uma página web simples e funcional sobre a empresa Keep Clean - Limpeza Profissional de Veículos.

O site apresenta informações completas sobre os serviços oferecidos pela empresa, incluindo galeria de imagens, vídeo demonstrativo e informações de contato, seguindo todos os requisitos técnicos solicitados.

Tecnologias Utilizadas

  • HTML5 semântico
  • CSS3 com estilos personalizados
  • Design responsivo
  • Incorporação de vídeo do YouTube
  • Estrutura otimizada para GitHub Pages
